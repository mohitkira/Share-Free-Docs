Welcome

Share Free Docs is a note sharing platform for teachers, student, and college. This project is more like a social networking site that can be used by faculty to share the record, study material, notice to the student. A student can also communicate with another student.

Installation guideline

1) download the sharefreedocs folder 2) download the sql file and install it. 3) edit the config file in function folder according to the database configuration. 4) then open the index.php file.

Admin Panel

You can open the admin panel using the following link: urwebsitename/sharefreedocs/admin/index.php login information: username: mohit password: mohit
